# Cu_PL_MOF-simulations

This repository contains videos of simulation of hexane isomers (2,2 DMB and 2,3 DMB) in the Cu2(bicyclo)2DABCO MOF.

## Contents

- compound1_22DMB.mp4 contains video of AIMD simulation of 2,2 DMB inside Cu2(bicyclo)2DABCO MOF.
- compound1_23DMB.mp4 contains video of AIMD simulation of 2,2 DMB inside Cu2(bicyclo)2DABCO MOF.
